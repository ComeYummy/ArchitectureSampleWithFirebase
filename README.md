# ArchitectureSampleWithFirebase

これは、iOSアプリを色々なアーキテクチャで実装してみたものです。
(詳しくは[こちらの記事](http://rock-name.com/ios/ios-architecture)で！)

Firebase(GoogleさんのmBaaS)でユーザーのSignUp/Login,投稿のCRUD処理をする、ものすごくシンプルなアプリです。
